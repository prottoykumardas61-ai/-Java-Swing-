##  Project Description
This is a simple Java Swing GUI application.  
It displays a window with two buttons: **Draw Circle** and **Clear**, and a drawing panel.

- Clicking **Draw Circle** draws a filled blue circle at a random position.
- Clicking **Clear** removes the circle and makes the panel blank again.

This project demonstrates:
- GUI development using Java Swing
- Event handling using ActionListener
- Custom drawing using paintComponent()

##  Technologies Used
- Java
- Java Swing (JFrame, JButton, JPanel)
- AWT (Graphics, Color)
- Event Handling (ActionListener)
